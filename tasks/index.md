## Stack Tecnológica

| Camada | Tecnologia |
| --- | --- |
| Framework | React + TypeScript |
| Canvas / Grafo | ReactFlow (`@xyflow/react`) |
| Estilização | Tailwind CSS |
| Estado global | Zustand |
| Formulários | React Hook Form + Zod |
| Serialização | JSON nativo |
| Testes | Vitest + React Testing Library |
